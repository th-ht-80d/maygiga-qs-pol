利用規約

第1条（適用）

本規約は、MyGiga（以下「本サービス」）を利用するすべての方に適用されます。本サービスの利用を開始した時点で、ユーザーは本規約に同意したものとみなされます。

第2条（利用条件と準拠契約）

本サービスは Apple が提供する Apple Media Services 利用規約 （Standard EULA） に準拠して提供されます。ユーザーは同規約および本規約の双方を遵守する必要があります。

第3条（自動更新サブスクリプション）

本サービスでは以下の自動更新サブスクリプションを提供します。価格は利用する国や通貨、キャンペーン状況によって変動する場合があり、購入確定前に App Store の画面で提示される金 額が適用されます。

プラン名 期間 主な特典
MyGiga プレミアム（月額） 1か月（自動更新） 保存期間 7日/14日に対応、保存容量拡張、広告非表示 ほか
MyGiga プレミアム（年額） 12か月（自動更新） 上記と同等の特典を年単位で提供、月額換算で割引価格

・課金は購入確認時に Apple ID に紐づく支払い方法へ請求されます。
・現在の利用期間が終了する24時間前までに自動更新をオフにしない限り、次回期間分の料金が更新時に請求されます。
・サブスクリプションの管理や自動更新の停止は、端末の「設定」>「Apple ID」>「サブスクリプション」から行えます。アプリ内からキャンセル操作を行うことはできません。
・利用期間中にサブスクリプションを解約した場合でも、契約期間の残日数分の返金は行われません。
・提供する機能や特典の内容は、事前の通知をもって変更または終了する場合があります。

アップロードされたファイルや通信環境等に起因する損害について、当社は故意または重過失がある場合を除き責任を負いません。本サービスは現状のまま提供され、特定目的への適合性や 無障害稼働を保証するものではありません。

第6条（サービスの変更・停止）

当社は、やむを得ない事情がある場合、ユーザーに事前通知のうえでサービス内容の変更、一時停止または終了を行うことがあります。重大な変更を実施する際はアプリ内通知や本ページの 更新を通じて周知します。

第7条（準拠法・裁判管轄）

本規約は日本法に準拠し、日本語版が優先して解釈されます。本サービスに起因または関連する紛争については、当社所在地を管轄する日本の裁判所を第一審の専属的合意管轄裁判所とし ます。

プライバシーポリシー

1. 収集する情報

  • サブスクリプション状態や購入履歴（Apple が提供する StoreKit 情報のみ）
  • アプリの利用ログ、操作イベント、クラッシュレポート（Firebase Analytics / Firebase Crashlytics）
  • 広告配信に必要な匿名化識別子、端末情報、概算の位置情報（Google AdMob）
  • 共有リンク生成時にユーザーが入力したフォルダ名や保存期間などのメタデータ
  • サポート問い合わせ時に任意提供された氏名・メールアドレス・スクリーンショット等

2. 利用目的

  • ファイル共有および保存期間の管理など、本サービスの提供・改善のため
  • 不正利用の監視、障害対応、セキュリティ向上のため
  • 課金状況の確認や購入サポートなどカスタマーサポート対応のため
  • 利用状況を分析し、機能開発や広告配信の最適化を行うため

3. 第三者提供・外部サービス

当社は、以下の外部サービス事業者へ必要な範囲で情報を提供し、各事業者のプライバシーポリシーに基づいて処理されます。

  • Apple（StoreKit、サブスクリプション決済）
  • Google LLC（Firebase Analytics / Crashlytics、Google Mobile Ads）
  • Cloudflare, Inc.（Cloudflare R2 ストレージ、共有データの保管）

4. 保存期間と削除

共有データは、ユーザーが設定した保存期間（最大14日）経過後に順次削除されます。法令に基づき保存が必要なデータを除き、目的達成後は速やかに消去します。

5. ユーザーの権利

ユーザーは、ご自身の個人情報の開示・訂正・利用停止・削除を要求できます。アプリ設定で広告トラッキングや通知の許可を変更できるほか、デバイスのオペレーティングシステム設定か ら分析/広告の許諾を制御できます。

6. セキュリティ対策

当社は、暗号化通信（HTTPS）の利用、アクセス制御、権限の最小化など、合理的な安全管理措置を講じて情報の漏えい・毀損を防止します。ただし、完全な安全性を保証するものではありま せん。

7. 改定

本ポリシーを改定する場合、変更内容を本ページに掲載し、重大な変更についてはアプリ内通知またはメールで周知します。改定後に本サービスを利用した場合、変更後の内容に同意したも のとみなされます。
